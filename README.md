## ABOUT THIS REST API

Well Its a basic REST API that enables you to do your basic CRUD applications.

All migrations have already been created, just add a database named : 'itemapi'

This Application was created with Laravel 8.0 the only major difference I noticed when putting this together was the route had to be more specific, I suspect that was one of the changes that came with the new version of laravel.
